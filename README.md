# GrowMint - Digital Marketing Agency Platform

A full-stack web application for a digital marketing agency built with PHP, MySQL, and JavaScript.

## Features
- User authentication (login/signup)
- Services showcase (SEO, Social Media, Paid Ads)
- Company listings page
- Contact form
- Dark mode toggle
- FAQ accordion

## Tech Stack
- PHP, MySQL (backend)
- HTML, CSS, JavaScript (frontend)

## How to Run
1. Set up XAMPP/WAMP
2. Import the database
3. Run on localhost
